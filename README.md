## How Does the Project Work?
### The website allows users to interact with a product list and a shopping cart system.
### With JavaScript, the application:
#### - Adds selected products to the cart.
#### - Calculates the total price automatically.
#### - Updates quantities and values in real time.
#### - Handles basic cart logic to simulate a real e-commerce experience.
